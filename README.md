# Knack App Schema

## 🚀 Available Scripts

- `blue-ball`: Create the blue-ball (linter + build + tests)
  - `lint`
  - `build`
  - `test`
- `start`: run the program

## 🥞 Dev Tech Stack

- `ts` (dev lang, transpiles to js)
- `eslint` (linter)
- `jest` (test runner)
